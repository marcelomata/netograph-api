CapSummary

A capture summary - this is the data that you will recieve for any query that

returns captures.

FieldTypeLabelDescription
id string

The globally unique ID for this capture.

assets string

The URL under which this capture's static assets can be found.

roots CapSummary.Root repeated

The roots for each URL in the capture plan.

time google.protobuf.Timestamp

The time of capture.

plan CapSummary.Plan

The capture plan.

stats CapSummary.Stats

Some basic statistics on the resulting capture.

FieldTypeLabelDescription

Roots that resulted from loading a URL in the the submitted plan. You can

think of this as the trajectory of redirections followed to reach the

final resting URL.

CaptureLogRequest

FieldTypeLabelDescription
dataset string

The name of the dataset to query.

limit int64

Limit the number of records that will be returned.

resume string

A resumption token, previously returned by an identical query.

start google.protobuf.Timestamp

The capture log is in reverse chronological order, so the start of the range is the most recent time. If start is zero, it's taken to be the largest possible time value.

end google.protobuf.Timestamp

The capture log is in reverse chronological order, so the end of the range is the least recent time. If end is zero, it's taken to be the smallest possible time value.
